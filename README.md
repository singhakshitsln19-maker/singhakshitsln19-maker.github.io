🚀 FRANCIS

Digital Utility Engine

Francis is a modern, matrix-inspired digital tools platform designed to deliver a fast, secure, and smooth web experience.

Live Website:
👉 "https://singhakshitsln19-maker.github.io/Francis/" (https://singhakshitsln19-maker.github.io/Francis/)

---

🌟 Features

- ⚡ Fast static website powered by GitHub Pages
- 🟢 Matrix-style animated background
- 📱 Progressive Web App (PWA) support
- 🔒 Service Worker for offline capability
- 🎨 Clean neon tech-inspired UI
- 📦 Installable on mobile devices

---

🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Service Worker
- Web App Manifest
- GitHub Pages
